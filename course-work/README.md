## Кратко описание на проекта
Проектът представлява цялостна система за управление на музей (Museum Management System). Позволява управление на основните дейности и ресурси на един музей. Система е разделена на две части:
1. **Backend (API):** Разработен с C# и ASP.NET Core, който управлява бизнес логиката и данните.
2. **Frontend (ReactFE):** Уеб интерфейс, разработен с React, който комуникира с API-то.

### Основни функционалности:
* **Потребители и роли:** Управление на потребителските профили и нивата на достъп (роли) в системата.
* **Изложби и колекции:** Каталогизиране и управление на предмети (items), колекции от предмети (item collections) и активни изложби (exhibits).
* **Пространства:** Организация на музейните зали (museum rooms) и добяване на зали и предмети към изложби.
* **Партньори и финансиране:** Управление на партньорски организации, изпращане на заявки за финансиране (fund requests).
* **Изследвания:** Управление на студенти, изпратени от партньорски организации, както и научни изследвания (studies), провеждани в музея от изследователи (researchers) и студенти.

## Структура на проекта
Проектът съдържа две основни папки:
* `MuseumManagmentSystem/` - Backend API (ASP.NET Core)
* `ReactFE/` - Frontend (React.js)

> **Важно:** За да работи клиентското приложение правилно, Backend API-то трябва да бъде стартирано и работещо, тъй като Frontend-ът извлича данни от него.

## Инструкции за инсталация и стартиране

### 1. Стартиране на Backend (API)
Backend приложението е разработено на C# (ASP.NET Core).

**Предварителни изисквания:**
* Инсталирано **Visual Studio** (за проекта е ползвана версия 2022) с пакет `.NET Web Development`.

**Стъпки за стартиране:**
1. Отворете папка `MuseumManagmentSystem` във **Visual Studio** (чрез `.sln` файла или като папка).
2. Намерете файла `Program.cs` в Solution Explorer.
3. Стартирайте проекта от бутона за старт в менюто или (ако компютъра ви е на издъхване като лаптопа ми) без дебъгер, като натиснете **`Ctrl + F5`**.
4. API ще се стартира и ще започне да слуша на локален порт.

---

### 2. Стартиране на Frontend (ReactFE)
Frontend приложението е разработено с React и изисква Node.js.

**Предварителни изисквания:**
* Инсталиран **Node.js** и **npm**.
* Инсталиран **Visual Studio Code (VS Code)**.

**Стъпки за инсталация и стартиране:**
1. Отворете папка `ReactFE` с **Visual Studio Code**.
2. Отворете вградения терминал в VS Code (`Ctrl + \`` или *Terminal -> New Terminal*).
3. Инсталирайте необходимите пакети и зависимости (изпълнява се само първия път):
   ```bash
   npm install
4. След като инсталацията приключи успешно, стартирайте приложението чрез терминала на VS Code с команда:
   ```bash
   npm start 
5. Приложението автоматично ще се зареди в браузъра ви.
